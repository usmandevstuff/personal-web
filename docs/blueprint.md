# **App Name**: AuraDev

## Core Features:

- Profile Display: Showcase developer's information, skills, and experience.
- Projects Showcase: Display personal projects with descriptions, links, and visuals.
- Contact Form: Enable visitors to send messages to the developer directly.
- Day/Night Mode Toggle: Allow users to switch between light and dark themes for optimal viewing.

## Style Guidelines:

- Color scheme: Dark, subdued purple-blue (#303841) for the background.
- Primary color: Muted teal (#77C4D3) for a techy feel that isn't cliche; to be used in components, highlights, and text.
- Accent: Golden yellow (#FFEA96) as a contrast color for key actions or information.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern look.
- Use modern, line-based icons for skills, projects, and contact information.
- Employ a clean, minimalist layout with plenty of white space to showcase content effectively.
- Incorporate subtle transitions and animations for a polished user experience.
